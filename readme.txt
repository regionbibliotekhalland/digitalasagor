Digitala sagor is a program for creating and displaying fairytales. It can be run in two modes:

Admin mode - used for creating and managing fairytales. Fairytales are created from images and sounds or videoclips. Admin mode is invoked by specifying command line argument admin. 

Playback mode - used for playing previously created fairytales. 

Before starting the program the inifile must be setup. See the user manual for more detailed information.
